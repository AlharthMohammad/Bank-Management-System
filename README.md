# Simple Bank System (C++)

## Description
A comprehensive console-based banking management system built using C++.  
This project simulates a real bank environment and includes client management, account operations, user administration, permissions, logging, secure authentication, and a full currency exchange module.

---

## Default Login Credentials
To access the system, use the following credentials:

- Username: `User2`  
- Password: `1234`

---

## Main Features

### Clients Management
- Add new client  
- Update client  
- Delete client  
- Find client  
- List all clients  
- View total balances  

### Transactions
- Deposit money  
- Withdraw money  
- Transfer between accounts  
- View transfer logs  

### Users Management
- Add new system users  
- Update user  
- Delete user  
- List all users  
- Login screen  
- Apply permissions per user  
- Register all logins into log file  

### Security
- Encrypt password before saving  
- Permission-based access control  

### Currency Exchange Module
- List currencies  
- Find a specific currency  
- Update currency rate  
- Currency calculator  
- Store and update currency data  

---

## Technologies Used
- C++  
- File handling (Text Database)  
- Object-Oriented Programming  
- Custom Headers & Classes  
- Basic Password Encryption

---



---

## How to Run
1. Open the project in Visual Studio or use g++ compiler  
2. Build the solution  
3. Run the program  
4. Log in using:  
   - Username: `User2`  
   - Password: `1234`  
5. Follow the on-screen menus to manage the bank

---

## Notes
- The system uses text files to store all data  
- Passwords are encrypted  
- Permissions restrict access to specific screens  
- All user logins and transfers are saved in a log file
